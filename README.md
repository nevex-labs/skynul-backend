# skynul-backend

Backend services for Skynul — multi-channel agent platform (Telegram, WhatsApp, Discord, Slack, Signal).

## Quick Start

```bash
pnpm install
pnpm dev
```

## Structure

```
├── server/        — Hono-based API server with WebSocket support
├── shared/        — Shared types and utilities
```

## Linting

```bash
pnpm lint       # check
pnpm lint:fix   # fix
```
